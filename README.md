# FTV Dashboard v68 — Gmail-inspired mobile app shell

Version corrigée pour rendre la version téléphone réellement utilisable :

- barre de recherche fixée en haut comme une app mobile ;
- burger fixé en haut à droite, visible et au-dessus de tout ;
- menu paramètres en vrai drawer au premier plan ;
- 5 onglets principaux en barre basse, avec icônes + labels courts ;
- rail horizontal des sports sous la barre de recherche pour accéder à tous les sports ;
- corps du dashboard scrollable sans chevauchements ;
- KPI plus compacts ;
- compétitions et vidéos en cartes mobiles ;
- graphiques/prévisions/analytics renforcés contre les cellules croquées ;
- desktop inchangé.

Structure Vercel Hobby conservée : une seule fonction API `api/[...path].js`.
