Tests & logs

Maintenance dashboard
Pour modifier le dashboard : remplace public/index.html, commit sur GitHub, Vercel redéploie avec le même domaine. La page client ne contient plus les clés sensibles.
Performance & snapshots automatiques
Le site utilise un snapshot serveur + cache CDN Vercel. Le client affiche les données préparées instantanément, puis actualise les vues publiques en arrière-plan.
